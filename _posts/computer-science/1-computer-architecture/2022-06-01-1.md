## HW SW

    - hw: CPU, Memory, Storage, Network

## 폰노이만 구조 

## 컴퓨터 구성 3 요소 
    1. CPU (Central Process Unit): 연산, 제어 
    2. Memory 
        - 내부 기억 장치: Register, Cache, DRAM 
        - 외부 기억 장치: SSD, HDD 
    3. IO device
        - Input: 마우스, 키보드
        - Output: 모니터, 스피커

    * Bus: CPU, Memory, IO device 를 연결하고 송수신해주는 장치
