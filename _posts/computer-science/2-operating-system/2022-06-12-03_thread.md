## Thread 
- 쓰레드란 
    - Light Weight Processe
    - 1 프로세스에 n 쓰레드 생성 가능 
    - 여러 CPU에 동시 실행 가능 
    - 프로세스의 데이터에 모든 쓰레드가 접근 가능 (Code, Data, Heap 영역 공유) 
    - 쓰레드마다 별도의 Stack 영역 소유 

* 프로세스는 다른 프로세스의 데이터 접근 불가 
* Multi Process와 Multi Thread 프로그래밍은 성능 개선을 목적으로 함
