# ORM: Object-Relation Mapping 

## 1. 개념 
Mapping 정보를 정의한 meta-data 를 사용해, Application Class 와 Database Table 에 자동으로(automated & transparent) 영속화 해주는 기술 

## 2. 배경 
- 도메인 모델 기반으로 코딩 가능 
- 객체지향 프로그래밍 가능 
- 코드 재사용성이 향상 
- 비즈니스 로직 구현 테스트가 쉬워짐
